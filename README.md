# Java Generating Theme App

Javaで作成したイラストのお題作成アプリです。

## 機能
- お題生成
- 難易度選択（4段階）
- 最新3件分の履歴保存、表示
- 最大10件のお気に入り保存、表示
- CSVファイル読み込み、書き込み
- 起動時の履歴・お気に入り復元

## 使い方
1. Odai.javaを実行
2. メニュー番号を入力
3. お題を確認
4. お気に入りに保存

## 学習内容
- Java基礎
- List（ArrayList、LinkedList）
- メソッド分割
- ファイル入出力
- CSVファイル操作
- Git/GitHub
